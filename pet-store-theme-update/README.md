# Pet Store Shopify Theme Starter

This is a beginner-friendly Shopify Online Store 2.0 theme starter for a pet store.

## What each folder does

- `layout` holds the main page wrapper.
- `templates` tells Shopify which sections appear on each page type.
- `sections` holds editable page blocks.
- `snippets` holds small reusable pieces of code.
- `assets` holds CSS, JavaScript, and images.
- `config` holds theme editor settings.
- `locales` holds text translations.

## Beginner learning path

1. Open `assets/theme.css`.
2. Change one color.
3. Save the file.
4. Upload the change to GitHub.
5. Let Shopify sync the repo.

Start with this line in `assets/theme.css`:

```css
background: #f7f9fc;
```

Change it to:

```css
background: white;
```
